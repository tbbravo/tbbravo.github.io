Bravo周刊投稿指南

Bravo周刊源码托管在github上，同时结合github的gh-pages能力和hexo飞一般的静态blog管理功能，让投稿者可以专注于用markdown完成文章内容，而不需要关心blog其他资源的管理，只需几行命令便可轻松搞定。噢耶，bravo！

安装git

已安装的用户请跳过此步骤。

  • windows用户
    Github Windows
  • mac用户
    Github Mac

    安装Node.js

    Node.js install

    安装hexo

新建文章

  • 拉取代码

  • 初始化hexo配置

  • 新建一篇文章

  • 编辑文章内容

  • 编译生成html

发布文章

  • 拷贝public文件

  • 提交master目录

预览效果

提交成功后，大概等上1分钟左右，访问这里查看效果

完成新文章的投稿后，需要提交原来在source分支下用markdown编写的内容
